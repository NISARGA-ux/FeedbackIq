🚀 FeedbackIQ

AI-Powered Feedback Insight & Comparison Tool

📌 Overview

FeedbackIQ is an AI-powered system that analyzes user feedback and converts it into actionable insights.

It helps teams:

Understand user sentiment

Identify recurring issues

Track improvements over time

Compare feedback across different periods

✨ Features
🔍 Feedback Analysis

Sentiment detection (positive / neutral / negative)

Keyword extraction

Issue identification

Suggestions & improvements

📊 Feedback Comparison

Compare two feedback inputs

Detect:

Improvements

New issues

Resolved problems

Generate actionable insights

⚡ AI-Powered Insights

Uses Groq LLM for fast inference

Converts raw feedback → structured intelligence

🛠 Tech Stack

Frontend: HTML / CSS / JS

Backend: Node.js

AI: Groq API

Runtime: Express / Fetch
